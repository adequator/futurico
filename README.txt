futurico README
==================

Приложение отдаёт json примерно следующего содержания:

	[
		[
			"Первый пост на этой борде",
			"Jovan"
		],
		[
			"Второй пост на этой борде",
			"Jovan"
		],
		[
			"Первый пост от сервера",
			"dirty"
		],
		[
			"Второй пост от сервера",
			"dirty"
		]
	]

Каждый запрос к странице обновляет счётчик в таблице counters.


f = lambda keys, values: {k: dict(zip(keys, values)).get(k, None) for k in keys}
